#Introduction
Sophia is a competitve open source tatical first person shooter, built on the open source Godot Engine. 
The main competitive game will consist of a 5v5 round based game mode. The Goal of the attacker team is to plant a bomb on a dedicated bomb side that the defender team has to protect. The attacker team has won, if the bomb explodes. Every round has a time limit of about 2 minutes. If the attacker team is not able to plant to eliminate the entire enemy team in the given time limit, the defende team has automatically won.
